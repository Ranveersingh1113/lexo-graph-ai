# How to Navigate to OAuth Consent Screen

## Current Location
You're on: **"OAuth overview"** page (Platform / Overview)

## Navigation Steps

### Option 1: Click "Audience" in Left Sidebar
1. Look at the **left sidebar** on your screen
2. You should see these menu items:
   - Overview (currently selected)
   - Branding
   - **Audience** ← Click this one!
   - Clients
   - Data access
   - Verification centre
   - Settings

3. Click **"Audience"** - this will show the OAuth consent screen where you can add test users

### Option 2: Use the Hamburger Menu
1. Click the **hamburger icon** (☰) at the top-left corner
2. In the expanded menu, search for or find **"APIs & Services"**
3. Click **"APIs & Services"**
4. Then click **"OAuth consent screen"**

### Option 3: Direct URL
1. Make sure "NLP project" is selected (or your project name)
2. Go to this URL directly:
   ```
   https://console.cloud.google.com/apis/credentials/consent?project=NLP-project
   ```
   (Replace "NLP-project" with your actual project ID if different)

## What You Should See After Navigation

Once you're on the OAuth consent screen, you should see:
- App name, user support email
- **"Test users"** section (this is what you need!)
- A button like **"+ ADD USERS"** or **"ADD TEST USERS"**

## Quick Visual Guide

From your current screen:
1. **Left sidebar** → Click **"Audience"** (3rd item down)
2. This opens the OAuth consent screen
3. Scroll down to find **"Test users"** section
4. Click **"+ ADD USERS"**
5. Add your email address
